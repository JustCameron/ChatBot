Informational Commands

Commands which might require a few parameters to give back useful information.

  • Help, "Assistance required in aisle 4!
    Usage: !help
  • Test, "Testing, testing! 1, 2, 3!"
    Usage: !test
  • Todo
    • <Default>, "What do we have here..?"
      ~~Displays the current state of the Todo list.
      Usage: !todo
    • Add, "We've got another one over here, Carl!"
      ~~Adds an item to the Todo list.
      Usage: !todo add [todo item]
      Parameters:
      • [todo item] --> String of text depicting what is to be done
      Examples: !todo add Make this page look bloody beautiful!
    • Edit, "Nah, this is wrong."
      ~~Edits an item on the Todo list.
      Usage: !todo edit [todo item index] [new item text]
      Parameters:
      • [todo item index] --> Index of the Todo item on the Todo list
      • [new item text] --> New text to be used for the Todo item
      Examples: !todo edit 3 Fix this one
    • Remove, "KILL IT WITH FIRE!!"
      ~~Removes an item from the Todo list.
      Usage: !todo remove [todo item index]
      Parameters:
      • [todo item index] --> Index of the Todo item on the Todo list
      Examples: !todo remove 5
Complex Commands

Commands perform complex operations requiring precise parameters to be utilised properly.

  • Roll, "Roll some dice or something idk"
    ~~Rolls some dice given an input string and outputs the results.
    Usage: !roll [roll request string]
    Parameters:
    • [roll request string] --> {number}{dice type} OR {number} (use + or - to add more dice)
    • {number} --> Any positive integer
    • {dice type} --> any positive integer preceeded by "d" e.g. d4, d6, d10
    Examples:
    • !roll d4 >> Rolls one 4-sided die
    • !roll 5d6 >> Rolls five 6-sided dice
    • !roll 3d8 + 4 >> Rolls three 8-sided dice and adds a constant value of 4
DnD Commands
  • DnD
    • Spell, "Cast some magic"
      ~~Get the details of a given Dnd 5e spell.
      Usage: !dnd spell [spell name]
      Parameters:
      • [spell name] --> Name of the spell you want to view
      Examples:
      • !dnd spell fireball
      • !dnd spell magic missile
    • CustomSpell
      • Create, "Create some custom magic!"
        ~~Being the process of creating a custom DnD spell.
        Usage: !dnd customspell create
      • Cancel, "Cancel some custom magic!"
        ~~Cancel the process of making a custom DnD spell.
        Usage: !dnd customspell cancel
      • Edit, "Edit some custom magic!"
        ~~Edits an attribute of custom DnD spell.
        Usage: !dnd customspell edit [spell name]
        Parameters:
        • [spell name] --> Name of the spell you want to edit
        Examples:
        • !dnd customspell edit 1H-KO Spell
        • !dnd customspell edit full party heal
      • Delete, "Delete some custom magic!"
        ~~Delete a custom DnD spell.
        Usage: !dnd customspell delete [spell name]
        Parameters:
        • [spell name] --> Name of the spell you want to delete
        Examples:
        • !dnd customspell delete loud spell of noise
        • !dnd customspell delete magical dust
      • View, "View some custom magic!"
        ~~View all of the custom DnD spells.
        Usage: !dnd customspell view
